# первый проект на Django
### О проекте
Это веб-приложение, разработанное на фреймворке **Django** (Python) в рамках лабораторной работы. Проект демонстрирует навыки создания серверной части сайта, работы с шаблонами и маршрутизацией.
### Реализованные разделы:
* **Главная страница (`/`)**: Центральный хаб магазина со ссылками на информационные разделы.
* **О магазине (`/info/`)**: Страница с описанием тематики лабораторной работы и специфики продаваемых товаров.
* **Об авторе (`/about/`)**: Персонализированная страница с данными студента (Тригорлова Анастасия, группа 88ТП).
